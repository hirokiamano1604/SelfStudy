# IELTS Writing: 書く前に確認すること

**目標: Band 7.5**

---

## 1. 問題文の読み方（2分）

- [ ] **何を聞かれているか** を正確に把握する（下線を引く）
- [ ] **指示語に注目**: discuss both views / agree or disagree / advantages and disadvantages / causes and solutions / to what extent
- [ ] **キーワードを特定**: 話題の範囲を限定する語に注目
- [ ] **Task 1 の場合**: 何のデータか、単位は何か、期間はいつか

---

## 2. 構成の型を決める（1分）

### Task 2 — Essay（250語以上）

| 問題タイプ | 構成 |
|---|---|
| Agree/Disagree | Introduction → 自分の立場を明示 → Body 1（主張+例） → Body 2（主張+例）→ Conclusion |
| Discuss Both Views | Introduction → View A → View B → 自分の意見 → Conclusion |
| Advantages/Disadvantages | Introduction → Advantages → Disadvantages → 自分の評価 → Conclusion |
| Causes & Solutions | Introduction → Causes → Solutions → Conclusion |
| Two-Part Question | Introduction → Part 1への回答 → Part 2への回答 → Conclusion |

### Task 1 — Report（150語以上）

| データ種類 | 構成 |
|---|---|
| グラフ/チャート | Introduction（言い換え） → Overview（全体傾向2-3点） → Detail 1 → Detail 2 |
| 地図 | Introduction → Overview（主な変化） → Detail 1 → Detail 2 |
| プロセス | Introduction → Overview（段階数・目的） → 前半の手順 → 後半の手順 |

---

## 3. 7.5に必要な4つの採点基準を意識する

### Task Response (TR)
- 問題の **すべての部分** に答えているか
- 自分の立場は **明確** か
- 各ポイントに **具体例・説明** があるか
- 一般論だけで終わっていないか

### Coherence & Cohesion (CC)
- 各段落に **明確なトピックセンテンス** があるか
- **論理的な流れ**: 主張 → 説明 → 具体例 → まとめ
- 接続詞の **バリエーション**（However / Nevertheless / On the other hand / In contrast）
- 接続詞の **使いすぎ** に注意（自然な流れ > 接続詞の羅列）

### Lexical Resource (LR)
- **同じ単語の繰り返し** を避ける（言い換えを用意する）
- **コロケーション** を意識する（make a decision, not do a decision）
- **アカデミックな語彙** を適切に使う（無理に難しい語を使わない）
- スペルミスは致命的 → 自信がない語は避ける

### Grammatical Range & Accuracy (GRA)
- **複文構造** を混ぜる（関係代名詞, 分詞構文, 仮定法, 倒置）
- **時制の一貫性**（Task 1: 過去データは過去形）
- **主語の多様性**: It is argued that... / Those who... / What is notable is...
- **冠詞・単複** に注意（最も多いミスのひとつ）

---

## 4. よくあるミスの自己チェックリスト

- [ ] Introduction で問題文をそのままコピーしていないか（**必ず言い換え**）
- [ ] Body段落が2つ以上あるか（1段落にすべて詰め込まない）
- [ ] 各Body段落にトピックセンテンスがあるか
- [ ] 具体例が抽象的すぎないか（"for example" の後に具体的な内容があるか）
- [ ] Conclusionで新しい情報を追加していないか
- [ ] 語数は足りているか（Task 1: 150語以上 / Task 2: 250語以上）
- [ ] Task 1 で意見を書いていないか（客観的な報告のみ）

---

## 5. 時間配分（30分練習用）

| フェーズ | 時間 | 内容 |
|---|---|---|
| 読解・構成 | 3分 | 問題分析、構成決定、キーアイデアのメモ |
| 執筆 | 20分 | 一気に書く、途中で悩まない |
| 見直し | 5分 | チェックリストに沿って確認 |
| 振り返り記録 | 2分 | 反省点・学んだ表現を記録 |

---

## 6. 書き始める前の3秒ルール

1. **「何を聞かれているか」を一言で言えるか？** → 言えなければ問題文を再読
2. **「自分の立場」は決まったか？** → 曖昧なまま書き始めない
3. **「各段落で何を言うか」の骨格はあるか？** → メモ3行で十分
