# IELTS Writing 練習プログラム

**目標: Band 7.5 | 1日30分 | 継続的改善サイクル**

---

## フォルダ構成

```
Writing/
├── README.md                  ← このファイル（全体ガイド）
├── PRE_WRITING_CHECKLIST.md   ← 毎回書く前に確認
├── templates/
│   ├── task1_template.md      ← Task 1 練習用テンプレート
│   └── task2_template.md      ← Task 2 練習用テンプレート
├── practice/
│   └── YYYY-MM-DD_task[1or2].md  ← 日々の練習ファイル
├── vocabulary/
│   └── word-bank.md           ← 語彙・表現の蓄積
└── review/
    └── scoring-sheet.md       ← 自己採点基準・振り返り
```

| 曜日 | タスク | 内容 | 重点ポイント |

| **日** | 語彙・表現整理 | word-bankの更新、表現の暗記 | インプットの強化 |

## 毎日の練習フロー（30分）

### フェーズ1: 準備（3分）
1. `PRE_WRITING_CHECKLIST.md` をざっと確認
2. 問題を読み、構成をメモする（3行程度）
3. テンプレートをコピーして `practice/` に日付ファイルを作成

### フェーズ2: 執筆（20分）
1. タイマーをセットして一気に書く
2. 途中で辞書を引かない、止まらない
3. 思いつかない表現は [ ] で囲んで後から対処

### フェーズ3: 自己レビュー（5分）
1. `review/scoring-sheet.md` の基準に沿って確認
2. 赤字（修正箇所）と青字（改善案）をメモ
3. 特に繰り返しているミスがないか確認

### フェーズ4: 振り返り記録（2分）
1. テンプレート下部の「振り返り」セクションに記入
2. 新しく学んだ表現を `vocabulary/word-bank.md` に追加
3. 次回の意識ポイントを1つだけ決める

---

## 改善サイクルの回し方

```
       書く
        ↓
    自己レビュー
        ↓
    弱点を特定
        ↓
   次回の意識ポイント設定
        ↓
     書く（意識して改善）
        ↓
       ...繰り返し
```

### 週次でやること
- [ ] 1週間の練習ファイルを通読し、共通するミスを3つ挙げる
- [ ] vocabulary/word-bank.md に最低10個の新表現を追加
- [ ] 来週の重点ポイントを1つ決める

### 月次でやること
- [ ] 月初と月末の作文を比較する
- [ ] scoring-sheet の点数推移を確認
- [ ] 重点ポイントを見直し、次月の課題を設定

## Band 7.5 到達の目安

| 基準 | 7.0 | 7.5（目標） |
|---|---|---|
| Task Response | すべての部分に回答、明確な立場 | + 一貫した深い議論、的確な例 |
| Coherence & Cohesion | 論理的な構成、適切なパラグラフ | + 自然な流れ、巧みな接続 |
| Lexical Resource | 幅広い語彙、一部ミスあり | + 正確なコロケーション、洗練された表現 |
| Grammatical Range | 複雑な構文を使用、一部エラー | + エラーが稀、多様な構文を自在に使用 |

---

## 始め方

1. まず `PRE_WRITING_CHECKLIST.md` を読む
2. `templates/` からテンプレートをコピーして `practice/` に配置
3. 問題を選んで書き始める（問題は自分で用意 or AIに生成してもらう）
4. 書き終わったら `review/scoring-sheet.md` で自己採点
