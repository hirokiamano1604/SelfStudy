# IELTS Writing フォーマット集

問題タイプごとに**構成の型**を固定する。練習時はこのフォーマットに従い、FBでは「フォーマット逸脱」を指摘して染み込ませる。

## 使い方

1. 問題を解く前に、該当する `task1_*.md` または `task2_*.md` を開く
2. 構成メモ・本文を書くとき、**段落の並びと必須要素**から外れないようにする
3. FB時に AI はこのフォーマットと照合し、逸脱があれば「フォーマット」として指摘する
