# Task 1 フォーマット：地図

**使う問題**: 同じ場所の「前」と「後」の地図の比較

---

## 段落の並び（厳守）

| 順 | 段落 | 役割 |
|---|---|---|
| 1 | Introduction | 何の地図か・どこ・いつ（before / after）の言い換え |
| 2 | Overview | 主な変化 2〜3 点（何が建った・撤去された・拡張した） |
| 3 | Detail 1 | 特定エリアまたは変化の種類ごとの具体（北側・中心部など） |
| 4 | Detail 2 | 別エリア・別の種類の変化 |

---

## 各段落の必須要素

### Introduction
- [ ] 場所・「2 つの時期」が入っている

### Overview
- [ ] 主な変化の方向（built / removed / expanded / replaced）を 2〜3 点
- [ ] 意見・推測を書いていない

### Detail 1 / 2
- [ ] 位置（north / center / next to）と変化を対応させている
- [ ] 時制は過去形で統一

---

## 禁止事項

- Overview 欠落
- 意見・理由の混入
- 地図にない施設・名前を創作する
