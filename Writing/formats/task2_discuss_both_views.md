# Task 2 フォーマット：Discuss Both Views + Opinion

**使う問題**: Discuss both these views and give your own opinion. / Discuss both views.

---

## 段落の並び（厳守）

| 順 | 段落 | 役割 |
|---|---|---|
| 1 | Introduction | 背景の言い換え + **両方の view の要約** + 自分の立場 |
| 2 | Body 1 | **View A**（問題文の 1 つ目の見方）の主張 + 理由 + 例 or So what |
| 3 | Body 2 | **View B**（2 つ目の見方）の主張 + 理由 + 例 or So what |
| 4 | Conclusion | 要約 + **自分の意見の再確認**（新情報を入れない） |

---

## 各段落の必須要素

### Introduction
- [ ] トピックの言い換え
- [ ] **両方の view** に触れている（Some people... / Others...）
- [ ] **自分の立場**が 1 文で分かる（I think... / In my view...）

### Body 1（View A）
- [ ] View A の主張がトピックセンテンスで明確
- [ ] 理由・具体例 or So what がある
- [ ] View B の主張を Body 1 で否定しすぎない（対比は Body 2 で）

### Body 2（View B）
- [ ] View B の主張がトピックセンテンスで明確
- [ ] 理由・具体例 or So what がある
- [ ] On the other hand / However 等で Body 1 と対比

### Conclusion
- [ ] 両方の view を**要約**
- [ ] 自分の立場の再確認
- [ ] **新しい論点・新しい例を入れていない**

---

## 禁止事項（フォーマット逸脱として指摘）

- 片方の view しか書いていない
- 自分の意見を Introduction か Conclusion のどちらかでしか述べていない
- Conclusion に新しい理由・例を書く
- Body 1 と Body 2 の役割が逆（View A と View B の対応がずれる）は内容の問題として指摘
