## 冠詞チェックリスト（書いた後に名詞を1つずつ確認）

### 3ステップ判定
```
Step 1: 数えられるか？ → No → 冠詞なし（education, music）
                        → Yes → Step 2
Step 2: 単数か？ → No（複数）→ 冠詞なしでOK
                  → Yes → 必ず a/an/the をつける → Step 3
Step 3: 特定できるか？ → No → a/an
                        → Yes → the
```

### よく間違えるパターン（暗記）
| パターン | 例 |
|---|---|
| be a/an + 職業 | be **a** doctor, become **an** artist |
| make a + 名詞 | make **a** decision, make **a** difference |
| have a + 形容詞 + 名詞 | have **a** decent amount, have **a** good time |
| the same / the only / the current | it is **the** same |
| a + 子音始まり / an + 母音始まり | **a** useful / **an** effective |

### 活動 vs 具体物（冠詞なし vs a/the）
| 冠詞なし（活動） | a/the あり（具体物） |
|---|---|
| enter **university**（大学生になる） | accepted by **a** university（ある大学に） |
| go to **school**（通学する） | go to **the** school（その学校の建物に） |

## 間接疑問文チェック
文中に how / what / whether / why が出てきたら → **主語 + 動詞** の語順にする

| ✗ 疑問文語順 | ✓ 平叙文語順 |
|---|---|
| what **is** he want | what he **wants** |
| how **is** society structured | how society **is** structured |
| whether **is** it valuable | whether it **is** valuable |
