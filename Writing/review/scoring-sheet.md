# IELTS Writing 自己採点シート

---

## 採点基準（Band 7.0–8.0 の違いを意識する）

### Task Response (TR)

| Band | 基準 |
|---|---|
| 6.0 | 関連する主要アイデアに対応しているが、一部不十分 |
| **7.0** | すべての部分に対応、明確な立場、主要アイデアが展開されている |
| **7.5** | + 一貫して深い議論、すべてのアイデアが十分にサポートされている |
| 8.0 | 十分に展開された回答、関連性のある具体的な例で完全にサポート |

**自己チェック:**
- 問題のすべての部分に答えたか → Yes / No
- 自分の立場は最初から最後まで一貫しているか → Yes / No
- 各主張に具体的なサポート（例・データ・理由）があるか → Yes / No
- 話が脱線していないか → Yes / No

---

### Coherence & Cohesion (CC)

| Band | 基準 |
|---|---|
| 6.0 | 情報やアイデアの配列は明確、接続詞を使用するが機械的 |
| **7.0** | 論理的な構成、明確な進行、適切なパラグラフ分け |
| **7.5** | + 自然な流れ、接続詞が目立たず効果的に機能 |
| 8.0 | シームレスな論理展開、段落間の接続が巧み |

**自己チェック:**
- 各段落にトピックセンテンスがあるか → Yes / No
- 段落間の論理的なつながりは明確か → Yes / No
- 接続詞が機械的に羅列されていないか → Yes / No
- 代名詞の指示対象は明確か → Yes / No

---

### Lexical Resource (LR)

| Band | 基準 |
|---|---|
| 6.0 | 十分な語彙、言い換えの試みあり、一部不正確 |
| **7.0** | 幅広い語彙、比較的柔軟に使用、稀にミス |
| **7.5** | + 正確なコロケーション、洗練された語彙選択 |
| 8.0 | 巧みな語彙使用、uncommon items を自然に使用、ミスはごく稀 |

**自己チェック:**
- 同じ単語を3回以上繰り返していないか → Yes / No
- 問題文の語彙を言い換えたか → Yes / No
- コロケーションは正確か → Yes / No
- 無理に難しい語を使っていないか → Yes / No

---

### Grammatical Range & Accuracy (GRA)

| Band | 基準 |
|---|---|
| 6.0 | 単文と複文の混合、一部エラーあり |
| **7.0** | 多様な複雑な構文、頻繁にエラーフリーな文 |
| **7.5** | + エラーが稀、多様な構文を自在に使用 |
| 8.0 | 幅広い構文、大多数がエラーフリー |

**自己チェック:**
- 単文ばかりになっていないか → Yes / No
- 複文構造を3種類以上使ったか → Yes / No
  - [ ] 関係代名詞（who/which/that）
  - [ ] 分詞構文（...ing / ...ed）
  - [ ] 仮定法（If... were/had）
  - [ ] 受動態
  - [ ] 倒置（Not only... but also）
  - [ ] 名詞節（What is important is...）
- 時制は一貫しているか → Yes / No
- 冠詞・単複のミスはないか → Yes / No

## 繰り返しミスのトラッキング

ここに自分が繰り返してしまうミスを記録する。3回同じタイプのミスが出たら、それは「癖」なので重点的に意識する。

| ミスの種類 | 具体例 | 発生回数 | 対策 |
|---|---|---|---|
| **冠詞の脱落（a/an/the）** | be musician → a musician / having gap year → a gap year | **24回**（2/22: 10、2/23: 7、2/24: 7） | 3ステップ判定: 数えられる?→単数?→特定? |
| **their のスペル** | thier → their | **3回**（2/22Re 1, 2/23Re 2） | the + ir で覚える |
| **interested のスペル** | intrested → interested | **2回**（2/23初+Re） | interest + ed |
| **beneficial のスペル** | benefitial → beneficial | **2回**（2/23Re） | benefit → beneficial（tをcに） |
| **it's / its の混同** | it's share → its share | 3回（2/21）→ その後出現なし | his にアポストロフィつけないのと同じ |
| **間接疑問文の語順** | what is he want → what he wants | 2回（2/23初）→ **2/23Reで修正済み ✓** | 文中の how/what/whether は平叙文語順 |
| **Conclusionに新情報** | 本文にない話をConclusionで展開 | 2回（2/20）→ **2/22で修正済み ✓** | Conclusion = 要約+立場再確認のみ |
| **Task 1 で意見を書く** | "This is because..." / "tradeoff" | 1回（2/21 初回）→ **リライトで修正済み ✓** | What だけ書く。Why は書かない |
| **leads to + 動名詞** | leads to maintain → leads to maintaining | 1回（2/23Re） | to は前置詞 → 後ろは動名詞 |
| **care about / relate to** | cared seriously → cared about / relates with → relate to | 2回（2/23Re） | 動詞+前置詞はセットで覚える |
| **use X into → use X for** | used 70% into agriculture → used 70% for agriculture | 2回（2/24初）→ **2/24Reで修正済み ✓** | use X **for** Y / allocate X **to** Y |
| **section vs sector** | three sections → three sectors | 4回（2/24Re） | section=文書の区画、sector=産業の区分 |
| **a half → half** | more than a half → more than half | 3回（2/24Re） | half は単独で使える。a 不要 |
| **the + 固有名詞** | the country A → Country A | 7回（2/24初）→ **2/24Reで修正済み ✓** | Tokyo に the をつけないのと同じ |
| **データ誤読** | Country B Industry 45%→40%と記載 | 2回（2/24Re） | 数字を書いたら必ず元データと照合 |
| **Task 1 で主観表現** | "It is interesting that" | 1回（2/24添削）| Notably / It is worth noting で代替 |
